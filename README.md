# ⚙️ Heritage Kitchen - Backend API

Il cuore pulsante di Heritage Kitchen, sviluppato in **Java** con **Spring Boot**. Gestisce la logica di business, l'autenticazione JWT e la persistenza dei dati su PostgreSQL.

---

### 🔗 Collegamenti del Progetto
* **🖥️ Frontend Repository:** [github.com/Enxos27/Heritage-Kitchen-Frontend](https://github.com/Enxos27/Heritage-Kitchen-Frontend) 
* **🚀 Live App:** [https://heritage-kitchen-front-end.vercel.app/](https://heritage-kitchen-front-end.vercel.app/)

---

### 🛠️ Tecnologie Utilizzate
* **Framework:** Spring Boot 3.x
* **Sicurezza:** Spring Security & JWT (Json Web Token)
* **Database:** PostgreSQL (ospitato su Koyeb)
* **ORM:** Spring Data JPA / Hibernate
